# Usage
<!-- usage -->
```sh-session
$ npm install -g swb
$ swb COMMAND
running command...
$ swb (--version)
swb/0.1.0 darwin-arm64 node-v23.10.0
$ swb --help [COMMAND]
USAGE
  $ swb COMMAND
...
```
